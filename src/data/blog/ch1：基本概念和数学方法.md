## 1.1 什么是优化问题？

**优化问题**就是在一定约束条件下，寻找使某个目标达到最优（最大或最小）的决策方案。

### 优化问题的三要素
- 决策变量；
- 目标函数；
- 约束条件

### 优化问题基本形式
$$
\begin{aligned}
max / \ min \quad f(x)\\
subject \, to \quad x \in \Omega
\end{aligned}
$$

### 可能的问答题
> **Q1**：请列举线性规划、整数规划、无约束优化各一个实际应用场景，并说明其三要素。
>
> **A1**：
> - **线性规划**——生产计划：变量是各产品产量，目标是利润最大化，约束是工时/原材料限制。
> - **整数规划**——背包问题：变量是选不选每个物品（0/1），目标是价值最大化，约束是背包容量。
> - **无约束优化**——最优定价：变量是商品价格，目标是收益最大化，无约束。

## 1.2 常见优化问题类型（问答题会考匹配）

|    问题类型    | 英文缩写 |      目标/约束特点       |      典型应用      |
| :--------: | :--: | :----------------: | :------------: |
|  **线性规划**  |  LP  | 目标线性 + 约束线性等式/不等式  | 生产计划、运输问题、资源分配 |
|  **整数规划**  |  IP  |  LP + 部分/全部变量取整数值  |  设施选址、排班、背包问题  |
| **混合整数规划** | MIP  | LP + 部分整数 + 部分连续变量 |  工厂开关决策、网络设计   |
|  **二次规划**  |  QP  |    目标二次 + 约束线性     | 投资组合优化、最小二乘拟合  |
| **无约束优化**  |  —   |     无约束，任意目标函数     |  机器学习训练、参数估计   |
|  **凸优化**   |  CP  |     凸目标 + 凸约束      | SVM、信号处理、控制系统  |
**关键记忆点**：
- LP是IP/MIP的松弛（去掉整数约束）
- QP比LP更一般（二次目标 vs 线性目标）
- 凸优化是所有优化中"最好解"的——局部最优=全局最优

## 1.3 数学基础（不单独出题，但会用到）

### 向量范数（计算必须会）
给定向量 $x = [x_1, x_2, \ldots, x_n]^\top$：

|      范数       |                  定义                   |   例子 $x=[2,3,-4]^\top$    |     |                 |
| :-----------: | :-----------------------------------: | :-----------------------: | --- | --------------- |
|   $L_1$ 范数    | $\|x\|_1$ = $\sum_{i=1}^{n} \|x_i\|$  |         2+3+(4)=9         |     | $2+3+4=9$       |
|   $L_2$ 范数    | $\|x\|_2 = \sqrt{\sum_{i=1}^n x_i^2}$ | $\sqrt{4+9+16}=\sqrt{29}$ |     |                 |
| $L_\infty$ 范数 |     $\|x\|_\infty = \max_{i} x_i$     |                           |     | $\max(2,3,4)=4$ |

**范数性质**：$\|x\| \geq 0$；$\|tx\| = |t|\|x\|$；三角不等式 $\|x+y\| \leq \|x\| + \|y\|$。

### 矩阵基础

- **秩（Rank）**：矩阵中线性无关行/列的最大个数
- **迹（Trace）**：对角线元素之和，$\text{tr}(A) = \sum a_{ii}$
- **行列式（Determinant）**：$|A|$，判断可逆性（$|A| \neq 0$ 则可逆）
- **逆矩阵**：$A^{-1}$，满足 $AA^{-1} = I$

### 函数梯度与Hessian

对于 $f(x_1, x_2, \ldots, x_n)$：
- **梯度**：$\nabla f(x) = \left[\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n}\right]^\top$（一阶导数向量）
- **Hessian矩阵**：$\nabla^2 f(x) = \left[\frac{\partial^2 f}{\partial x_i \partial x_j}\right]_{n \times n}$（二阶导数矩阵，对称）

> **例题（hw1 习题2.4）**：$f(x_1, x_2) = ax_1^2 + bx_1x_2 + cx_2^2$，求梯度和Hessian，并问Hessian正定的条件。
>
> **解答**：
> $$\nabla f(x) = \begin{bmatrix} 2ax_1 + bx_2 \\ bx_1 + 2cx_2 \end{bmatrix}, \quad \nabla^2 f(x) = \begin{bmatrix} 2a & b \\ b & 2c \end{bmatrix}$$
> 正定条件：$2a > 0$ 且 $4ac - b^2 > 0$，即 $a > 0$ 且 $4ac > b^2$。
