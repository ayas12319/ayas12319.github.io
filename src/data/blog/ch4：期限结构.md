# 收益率曲线与期限结构
## 收益率曲线
收益率与期限的关系称作**收益率曲线**。

## 期限结构
**期限结构：**
在某个时刻，不同时间长度(期限)的即期利率所组成的曲线。

# 即期利率
**即期利率$s_{t}$：**
以年的形式表示，是从开始(t = 0)到时间t持有货币收取的利率。
即$s_{t}$表示持有t年的货币支付利率。t年后，存款以$(1+s_{t})^{t}$的因子增长。
**每一个期限的即期利率实际上就是等于对应期限零息债券的收益率**

**NOTE:（与到期利率的区别）**
到期利率更像是对于市场利率的平均估计，而即期利率反应的是市场价格变化的真实反映。

**现值：**
- 贴现因子$d_{t}$：计算在时刻t的支付的现值
- 现金流的现值PV = $x_{0} + d_{1}x_{1} + ... + d_{n}x_{n}$

**即期利率的确定：**
- 递归方法确定：
    例如首先确定$s_{1}$，其次再通过方程：$P = \frac{C}{1 + s_{1}} + \frac{C + F}{(1 + s_{2})^{2}}$解出$s_{2}$，类似地依次解出对应的值。
- 使用不同息票率相同期限的债券**构造零息债券**，从而确定即期利率。
    例题：
    ![](https://cdn.jsdelivr.net/gh/ayas12319/picture-bed@main/Snipaste_2026-03-25_21-53-46.png)
    构造一个前九年的现金流为0的债券组合，只有在第10年才有现金流产生。
    故设分别买入x，y份A，B债券，则在前九年的现金流为：$10x + 8y$，令其等于0，则解得x = -4， y = 5。故在第10年得到：108 * 5 - 110 * 4 = 100。
    又因为买入的成本为：$5 * 85.89 - 4 * 98.72 = 34.57$。故10年的即期利率为满足等式：$34.57 * (1 + s_{10})^{10} = 100$

# 远期利率
**远期利率：**
现在达成协议的未来两个日期之间的利率，即$f_{t_{1}, t_{2}}$，$t_{1} < t_{2}$。

## 套利论据
设f表示一年后银行的借贷利率。
对于情况：
- 第一年，以利率s1借入1元\[到期时需要偿还给银行(1 + s1)元\]，将1元以两年利率s2存入银行。
- 第二年，以远期利率f借入(1 + s1)元，用于还债。
若存在$(1+s1)(1+f) < (1+s_{2})^{2}$，则说明存在套利机会。

## 远期利率公式
根据套利公式一般相等得到：
- 按年复利：$(1 + s_{i})^{i}(1 + f_{i,j})^{j-i} = (1+s_{j})^{j}$
- 每年m期复利：$\left(1 + \frac{s_i}{m}\right)^{i m} \cdot \left(1 + \frac{f_{i,j}}{m}\right)^{(j-i) m} = \left(1 + \frac{s_j}{m}\right)^{j m}$
- 连续复利：$e^{s_{t2}t_{2}} = e^{s_{t1}t_{1}}e^{f_{t1,t2}(t2-t1)}$
# 对利率期限结构的解释
- 流动性偏好理论：相对于长期证券，投资者通常更偏好短期固定收益证券。
- 市场分割理论：市场根据到期期限彼此分割，由各自的市场供需力量决定。
- 预期理论：即期利率由未来的利率期望值决定。

# 期望动态
## 即期利率预测
由远期利率可以得到，对于下一年即期利率的预测有：
$$
s_{j-1}' = f_{1,j} = [\frac{(1+s_{j})^{j}}{(1+s_{i})}]^{\frac{1 }{ j-1}} - 1
$$
且可以得到，对于每一轮的预测数目都会逐渐减少1年，故总体呈一个倒三角的样式。

## 贴现因子
可以使用$d_{j,k}$表示将时间点k收到的现金贴现成时间点j的相应金额的贴现因子。
贴现因子可以使用远期利率来表示：
$$
d_{j,k}=[\frac{1}{1+f_{j,k}}]^{k-j}
$$
## 短期利率
**短期利率即是跨度为1年的远期利率。**

# 期限结构下的久期
在期限结构框架中，收益率不是一个固定量，故考虑即期利率曲线的平行移动：对于即期利率$s_{1}, s_{2}, s_{3},...,s_{n}$，设新的即期利率为$s_{1} + \lambda, s_{2} + \lambda ,..., s_{n} + \lambda$。

**费雪-威尔久期**
**对于连续复利计息**，其现值计算为：
$$
PV = \sum\limits_{i=0}^{n}x_{t_{i}}e^{-t_{i}(s_{t_{i}}+ \lambda)}
$$
价格的敏感度为：
$$
\frac{dP(\lambda)}{d\lambda}|_{\lambda = 0} =- \sum\limits_{i=0}^{n}t_{i}x_{t_{i}}e^{-s_{t_{i}}t_{i}}
$$
定义费雪-威尔久期为：
$$
D_{FW} = \frac{1}{PV}\sum\limits_{i=0}^{n}t_{i}x_{t_{i}}e^{-s_{t_{i}}t_{i}}
$$

所以有：
$$
D_{FW} = -\frac{1}{P(0)}\frac{dP(0)}{d\lambda}
$$
**离散时间复利计息**
按照每年复利计息m次来计算，设第k期的即期利率为$s_{k}$，有现金流$(x_{0},x_{1},...,x_{n})$，则对应的价格为：
$$
P(\lambda) = \sum\limits_{k=0}^{n}x_{k}(1+\frac{s_{k} + \lambda}{m})^{-k} 
$$
故得到：
$$
\frac{dP(0)}{d\lambda} = \sum\limits_{k=1}^{n}-(\frac{k}{m})x_{k}(1 + \frac{s_{k}}{m})^{-(k+1)}
$$
两边同样同时除以-P(0)得到对应的久期概念：
$$
D_{Q} = -\frac{1}{P(0)}\frac{dP(0)}{d\lambda}
$$
称$D_{Q}$为**半修正久期**，其给出了关于即期利率曲线平移的相对价格敏感度
