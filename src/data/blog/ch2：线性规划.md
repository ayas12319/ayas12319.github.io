# 线性规划标准形式
线性规划问题的标准型如下：
$$
\begin{aligned}
min \quad c^{T}x \\
s.t. Ax = b \\
x >= 0
\end{aligned}
$$
其中x$\in R^{n}$，A是m * n 矩阵，b $\in R^{m}$
**（n是问题的特征维度数量，m是约束的数量）**
- 所有的线性规划问题都可以转化为上述形式。
- 通常假设A为行满秩(m < n)。
    若m = n，则有唯一解；若m > n，则方程无解。

## 各种情况转标准型的方法

| 原问题特征 | 转换方法 | 说明 |
|:----------:|:--------:|:----:|
| 目标是 $\max$ | 变为 $\min (-c^\top x)$ | 最大化利润 = 最小化负利润 |
| 约束是 $\leq$ | 左边 + 松弛变量 $s \geq 0$ | $x_1 + 2x_2 \leq 6$ → $x_1 + 2x_2 + s = 6$ |
| 约束是 $\geq$ | 左边 - 剩余变量 $s \geq 0$ | $x_1 + 2x_2 \geq 6$ → $x_1 + 2x_2 - s = 6$ |
| 等式约束 | 保持不变，或引入人工变量 | 见大M法/两阶段法 |
| 变量 $x_j \leq 0$ | 令 $x_j = -x_j'$，$x_j' \geq 0$ | 替换后新变量非负 |
| 变量 $x_j$ 无约束 | 令 $x_j = x_j^+ - x_j^-$，两者 $\geq 0$ | 拆成正负两部分 |

# 例子：
## 生产问题

$$
\begin{aligned}
设产品I，产品II的生产数量为x_{1}, x_{2}\\
目标函数：max \quad 6x_{1} + 8x_{2} \\
constraint :\quad 4x_{1} + 4x_{2} + d_{1} &= 40 \\
5x_{1} + 10x_{2} + d_{2} &= 60 \\
x_{1}, x_{2}, d_{1},d_{2} &>= 0
\end{aligned}
$$
