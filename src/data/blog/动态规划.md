**动态规划核心思想**
基于最优子结构特征，自底向上地计算最优解，并在过程中，记住所计算的子问题的答案，从而降低时间复杂度

# 动态规划求解矩阵链乘问题步骤：
Step1：寻找子问题空间
对于任意的$1 \le i \le j \le n$，将$A_{i}, A_{i+1},...,A_{j}$的最优化括号方案定义为原问题的一个子问题。

Step2：寻找最优子结构
假设$A_{t}$有$p_{t-1}$行$p_{t}$列。
- 当i < j时，假设$A_{i...j}$的最优括号化方案在$A_{k}$和$A_{k+1}$之间分裂，即最后一次矩阵相乘为：$A_{i...k} * A_{k+1,...,j}$
- 则$A_{i...j}$的最优方案必然是$A_{i}$，$A_{i+1},...,A_{k}$的最优括号方案计算$A_{i,...k}$，用$A_{k+1},A_{k+2},...,A_{j}$的最优括号化方案计算$A_{k+1,...,j}$。再将两者相乘。

Step3：确定最优解的递归式
设$m[i,j]$为计算子问题$A_{i,...j}$的最小计算成本。
则递归式为：
$$
m[i,j] = \left\{\begin{matrix}
 0 &i=j \\
  min\{m[i,k] + m[k+1.j] + p_{i-1}p_{k}p_{j}\}& i < j
\end{matrix}\right.
$$

Step4：根据递归式，自底向上计算最优解的值
- 算法输入：$p = (p_{0},p_{1},....,p_{n})$：其中$p_{i-1} * p_{i}$是矩阵$A_{i}$的维数
- 主要结构：
二维矩阵m：m[i, j]记录$A_{i,...j}$的最优解所产生的标量惩罚成本
二维矩阵S：$S[i,j]$记录$A_{i,...j}$的最佳分裂点。

# 带有权重的区间调度问题
目标为：选择具有最大权重和的子集。

首先按照结束时间排序：
定义$p(j)$= 最后一个结束时间不晚于工作j开始时间的工作编号。

定义$OPT(j)$ = 由工作$1,....,j$组成的集合中所选择的最大权重。
- Case1：OPT选择了工作j：
1，则不能选择与工作j冲突的：$\{p(j)+1, p(j)+2,...,j-1\}$
2，必须包括工作$1,2,...,p(j)$组成的一个最优解——$OPT(p(j))$

- Case2：没有选择工作j:
即必须包括了工作$1,2,...j-1$的一个方案——$OPT(j-1)$

递推式为：
$$
OPT(j) = \left\{\begin{matrix}
 0 &j=0 \\
  max\{v_{j} + OPT(p(j)), OPT(j-1)\}& j \neq 0
\end{matrix}\right.
$$

# 最小二乘
线性回归：
- 给定平面中的n个点
- 找到一个直线$y = ax+b$，最小化平方误差
$$
SSE = \sum\limits_{i=1}^{n}(y_{i}- ax_{i} - b)^{2}
$$
设置目标函数为：$E + cL$——E为平方误差和，c为一个常数，L为使用线段的个数。

- OPT(j)：用前面j个点使得目标函数最小的方案。
- $e(i, j)$：$p_{i...j}$的最小平方误差和。
故得到递推式为：
$$
OPT(j) = \left\{\begin{matrix}
 0 &j=0 \\
  min_{1 \le i \le j}\{e(i,j) + c + OPT(i-1)\}& j \neq 0
\end{matrix}\right.
$$
伪代码：
```
INPUT: n, p1...pN, c

Segmenter-Least-Squares() {
  M[0] = 0 
  for j = 1 to n
    for i = 1 to j
      compute the e[i][j] for the p_{i...j}
  for j = 1 to n
    M[j] = min_{1 <= i <= j}(e[i][j] + c + M[i-1])
  return M[n]
}
```
现时间复杂度为$O(N^{3})$

利用前缀和进行优化：
可使用O(N)的时间计算前缀和来计算e(i, j)。

# 背包问题

定义子问题空间：

若定义OPT(i) = 由$1....i$物品组成的最大收益子集。
- case1：OPT不包括物品i
    opt(i) = opt(i-1)
- case2：OPT包括物品i
    在目前定义的状态量下，无法反映当前是否满足选择物品i的状态。

定义OPT(i, w) = 在限制最大重量为w的前提下，由物品$1....i$组成的最大收益子集。
- case1：OPT不包括物品i
    opt(i, w) = opt(i-1, w)
- case2：OPT包括物品i
    OPT为在重量限制为$w - w_{i}$的前提下，由物品$1...i-1$组成的最大收益子集

对应的递推式为：
$$
OPT(i, w) = \left\{\begin{matrix}
 0 &i=0 \\
  opt(i-1,w)& w_{i} > w \\  
 max\{OPT(i-1, w),  v_{i} + OPT(i-1, w-w_{i})\} & otherwise
 \end{matrix}\right.
$$
伪代码实现：
```
INPUT: n, w[n], v[n]
for w = 0 to W
    M[0, w] = 0
for i = 1 to n 
  for w = 1 to W
     if(w[i] > w)
       M[i, w] = M[i-1, w]
     else
       M[i, w] = max(M[i-1, w], v[i] + M[i-1, w-w[i]])
return M[n, W]
```

运行时间：O(nW)——即二重循环

# RNA二级结构
一个碱基对集合满足以下条件：
- 集合S中的每一对碱基都是互补配对
- 每对配对碱基之间至少间隔4个碱基
- 无交叉：若$(b_{i}, b_{j})$和$(b_{k}, b_{l})$都是S中的配对，则不能出现：$i < k < j <l$

目标：
给定一串RNA分子，求一个二级结构S，使得碱基配对数量最大。

子问题空间定义：
定义OPT(i, j) = 从子串$b_{i}....b_{j}$中能够产生的最大配对碱基对数量。
- Case1：若$i \ge j - 4$，子串过短，无法配对
- Case2：不包括末尾碱基$b_{j}$：
    OPT(i, j) = OPT(i, j-1)
- Case3：末尾碱基$b_{j}$与前面的某个$b_{t}$配对（$i \le t \le j-4$）
    OPT(i, j) = 1 + $max_{t}OPT(i, t-1)+OPT(t+1, j-1)$

伪代码：
```
RNA(b1,....,bn){
 for k = 5, 6, n-1 //枚举区间长度
   for i = 1, 2, ...,n-k //枚举区间起点
     j = i + k
     compute M(i, j)
 return M(1, n)
}
```

# 序列对比
给定两个字符串A, B，假设A是B的错误拼写，则A与B 之间的编辑距离定义为：
- 设空位罚分：$\delta$即插入/删除一个字符的代价
- 设错配罚分：$\alpha_{pq}$即字符p和q不匹配的代价
- 总代价 = 所有空位罚分和错配罚分之和。

问题目标：
给定两个字符串：$X = x_{1}x_{2}...x_{m}$和$Y = y_{1}y_{2}...y_{n}$，找到将X改造为Y的最小操作次数，即最短编辑距离。

则总代价可以表示为：
$$
cost(M) = \sum\limits a_{x_{i}y_{j}}(mismatch) + \sum\limits\delta_{x}+ \sum\limits \delta_{y} (gap)
$$

定义子问题空间：
OPT(i, j) = 子串$x_{1}...x_{i}$和$y_{1}...y_{j}$相匹配的最小花费。
- Case1：$x_{i}$与$y_{j}$配对：
    $x_{i}$与$y_{j}$的错配代价+OPT(i-1, j-1)
- Case2：$x_{i}$没有配对
    $x_{i}$的gap代价+OPT(i-1, j)
- Case3：$y_{j}$没有配对
    $y_{j}$的gap代价 + OPT(i, j-1)

得到递推式为：
$$
OPT(i, j) = \left\{\begin{matrix}
  j\delta& i=0\\
  min(\alpha_{x_{i}y_{j}}+OPT(i-1,j-1), \delta + OPT(i-1,j), \delta+OPT(i, j-1))& otherwise\\
  i\delta& j=0
\end{matrix}\right.
$$

伪代码实现：
```
f(){
 for i = 0 to m 
  M[0, i] = i delta
 for j = 0 to n
  M[j, 0] = j delta
 for i = 1 to m 
   for j = 1 to n
     M[i, j] = min(alpha[x_i, y_j] + M[i-1, j-1], delta + M[i-1, j], delta +   M[i, j-1])
     
 return M[m, n]
}
```
时间复杂度为：O(mn)
空间复杂度为：O(mn)
