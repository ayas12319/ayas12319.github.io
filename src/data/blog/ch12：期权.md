# 期权基础
## 期权定义
**期权是授予持有人一项在未来确定的一段时间内以确定的价格购买或出售确定资产的权利。**
- 期权不是免费的，买方需要先支付一笔费用给卖方。
- 对期权买方来说，合约赋予的只有权利而无义务
- 对于期权卖方来说，合约赋予的只有义务而无权力。

## 期权内容
- 标的资产的种类及数量
- 执行价格
- 合约有效期限
- 期权交易地点
- 权利金：购买期权所支付的费用
- 合约格式：规定合约的交易单位，最小变动价格，每日最高波动幅度，合约月份等。

## 期权术语
- Buy/Long：期权合约的购买者
- Sell/Short：期权合约的出售者

- Call Option：看涨期权/买方期权——期权出售者基于期权持有者在未来某段时间以执行价格购买标的资产的权利。
- Pull Option：看跌期权/卖方期权——.....出售标的资产的权利

- 实值期权：执行期权有利：
    - Call：市场价格 > 执行价格
    - Put：执行价格 > 市场价格

- 虚值期权：执行期权不利

- 平值期权：执行价格和市场价格相等

- 美式期权：期权可以在到期日前任何时刻执行
- 欧式期权：期权只能在到期日执行

# 期权的到期价值
## 买入看涨期权的到期价值
- 若股票价格为$S_{T}$，执行价格为$K$
- 买入看涨期权的支付：
$$
Payoff = \begin{cases}
  S_{T} - K& if \, S_{T} \ge K \\
  0& if \, S_{T}< k
\end{cases}
$$
- 买入看涨期权的利润 = Payoff - 期权费

## 卖出看涨期权的到期价值
卖出看涨期权的支付：
$$
Payoff = \begin{cases}
  -(S_{T} - K)& if \, S_{T} \ge K \\
  0& if \, S_{T}< k
\end{cases}
$$
卖出看涨期权的利润 = Payoff + 期权费
![](https://cdn.jsdelivr.net/gh/ayas12319/picture-bed@main/Snipaste_2026-06-09_16-04-34.png)
## 买入看跌期权的到期价值
买入看跌期权的支付：
$$
Payoff = \begin{cases}
  0& if \, S_{T} \ge K \\
  K - S_{T}& if \, S_{T}< k
\end{cases}
$$
买入看跌期权的利润 = Payoff - 期权费

![](https://cdn.jsdelivr.net/gh/ayas12319/picture-bed@main/Snipaste_2026-06-09_16-06-55.png)
## 卖出看跌期权的到期价值
卖出看跌期权的支付：
$$
Payoff = \begin{cases}
  0& if \, S_{T} \ge K \\
  -(K - S_{T})& if \, S_{T}< k
\end{cases}
$$
卖出看跌期权的利润 = Payoff + 期权费


# 期权与股票投资
股票期权价格取决于标的股票价格。标的股票变动不仅影响其自身收益，同时间接影响股票期权的收益。
## 期权交易与股票交易比较
- 杠杆作用：期权对于股票价值变动的敏感性更大。
- 潜在保险功能：期权可以帮助投资者保证最低收益率，或限制最大损失。
