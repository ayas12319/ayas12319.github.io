## 光标移动
h向左移动；j向下移动；k向上移动；k向右移动；
## 插入文字
- 光标移动到指定位置，点击i切换为INSERT模式，输入对应字符，再按ESC切换回NORMAL模式。
- ce删除并替换对应的单词进入INSERT模式；cc对于整行执行相同的功能
## 句尾添加文字
光标移动到对应的行，点击A，输入对应字符，再按ESC切换回NORMAL模式。
## 修改文件
使用vim file.txt打开对应的文档文件，在其中正常修改
最后使用:wq< ENTER >进行文件的保存和退出vim模式。
## 退出VIM模式
- :q! 丢弃所有更改，直接退出
- :wq 保存所有更改，退出VIM 模式
# 删除
- dw删除至下一个单词前
- de删除至此单词的末尾
- d$删除至本行的末尾
- dd删除整行
- 2dd删除两行
- x删除单个字符

# 便捷移动：
2w移动到两个单词后；
3e移动到第三个单词后；
0移动到行首位置。
基本格式：
操作 + 操作次数 + 对文本的操作长度：w指单词，e指单词的末尾，$指本行的结尾。
# 撤销操作：
u撤回上一个操作；使用U使整行撤回原有操作；
CTRL + R重新执行上一个操作
# PUT指令：
在某行使用dd删除该行，并保存到vim寄存器中。
在光标处使用p，在光标下方粘贴该内容。
# 替换指令：
在错误处的前面使用r，再敲入对应想要修改为的字符。
基本模式：
c + number + motion
motion相同，即是w表word，$表整行的结束

# 查看当前位置和文件属性
输入CTRL + G显示当前所在行号和文件属性；
输入G跳转到文件末尾；
输入gg跳转到文件开头；
输入行号和G跳转到指定的行段。

# 搜索指令：
1，输入 / 进入搜索模式，之后输入想要搜索的单词
2，输入n，搜索下一个匹配项；输入N，搜索上一个匹配项
3，输入?进入反向搜索，此时n与N功能相反。
4，ctrl + o返回上一个位置，CTRL + i向下一个跳转
# 匹配括号指令：
在目标(, ), {, }, ......前输入%，自动跳转到相匹配的括号前。
# 替换指令：
- :s/原单词/新单词 转换第一个匹配项
- :s/原单词/新单词/g，整行进行替换
- 指定替换范围—— :A, Bs/原单词/新单词/g
- 全文件替换——:%s/原单词/新单词/g
# 外部终端命令：
：进行一般命令模式；
:! 后接对应终端命令，在外部终端窗口中显示结果。

# 保存vim文件：
:w + 对应文件名——将当前vim编辑器中的内容保存到对应文件中；
:del + 对应文件名——删除对应文件
## 保存指定内容：
使用v进入单字符可视模式，移动光标进行选中。
输入:，在后面输入w + 保存文件名。
NOTE：使用V进行行选中模式。CTRL + v进行矩阵块选中模式。

# 读取合并文件：
- 使用:r 文件名，将指定文件内容粘贴到对应光标下方。
- 也可以读取外部指令输出：
    - 例如 :r !ls     将文件列表复制到下方；
    - :r !date 将日期复制到下方；

# 添加命令：
1，输入o，在下方开新行；输入O，在上方开新行，同时自动进入INSERT模式
2，追加命令：
- 使用0或^移动到该行开头；
- 使用e移动到单词末尾；

# 复制粘贴命令：
按v进入可视模式后，选定内容，按y复制内容；
再使用p粘贴内容。

# 设置选项的配置：
1，设置忽略大小写：set ic
2，设置搜索高亮和增量搜索：set hls is
3，关闭忽略大小写：set noic

# 获取帮助文档：
1，按< F1 >打开帮助文档，或:help
2，使用ctrl + w 切换编辑窗口与帮助窗口
3，:q关闭帮助窗口
4，创建编辑某个文件：:e ~/\_vimrc
5，提示相关命令，例如输入:e后，ctrl + d显示以e开头的指令。
6，提示文件名：输入xxx后，输入tab会显示已有的匹配文件。
