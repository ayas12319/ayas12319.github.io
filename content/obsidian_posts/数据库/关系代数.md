# 八种操作：
**传统操作集合：**
- 并
- 交
- 差
- 迪卡尔积
**特殊关系操作：**
- 选择
- 投影
- 连接
- 除
封闭性：任何给定关系运算的输出都是另一个关系
**笛卡尔积：**
A TIMES B = A与B的笛卡尔积
结果的基数等于两者基数之积
结果的度等于两者之和
**投影：**
关系A在属性X,Y,....上的投影：
A{X, Y, .....}
其结果是一个关系：其表头为 {X, Y, …, Z}，其主体由所有形如 {X:x, Y:y, …, Z:z} 的元组集合构成，使得在关系 A 中存在一个元组，其 X 值为 x，Y 值为 y，…，Z 值为 z。相当于SELECT语句。
**自然连接—JOIN：**
A与B的自然连接：A JOIN B
其结果是一个关系，其表头为 {X, Y, Z}，其主体由所有形如 {X:x, Y:y, Z:z} 的元组集合构成，这个关系在A与B中均存在
**除法运算：**
A DIVIDEBY B PER C
设关系 A、B 和 C 的表头分别为 {X}、{Y} 和 {X, Y}：
返回在 **C** 中那些与 **B** 中所有元素都在 **A** 中有联系的元素。
