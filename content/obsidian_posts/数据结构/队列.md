# 链表模拟队列实现：

```c++
template <class Type> struct node{
   Type data;
   node* link;
};

template <class Type> 
class LinkedQueue{
   private:
     node<Type>* head, tail;
   public:
     LinkedQueue(){
        head = tail = nullptr;
     }
     
     void enqueue(Type x){
        node<Type>* temp;
        temp = new node<Type>();
        temp->data = x;
        temp->link = nullptr;
        if(head == nullptr) head = temp;
        else tail->link = temp , tail = temp;
     }
     
     Type dequeue(){
        if(head == nullptr) return INT_MIN;
        Type ans = head->data;
        node<Type>* temp = head;
        head = head->link;
        delete temp;
        return ans;
     }
}

```

# 循环队列：

即队头，队尾的加1是实现从maxSize-1到0之间的循环。通过取模运算实现：  
队满条件： （rear + 1) % maxSize = front  
**基本操作：**

- 出队操作：（队头+1）% maxsize；
- 入队操作：（队尾 + 1）%maxsize

# 双向队列：

- 允许在两端进行插入和删除操作的线性表结构，用两个结点left和right来表示左端结点和右端结点。
- 双向队列为空的标志：left > right;
- 初始位置应该选在中间——(Maxn - 1) / 2
